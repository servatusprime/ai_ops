# licenses_disclaimers

## Purpose

Store reusable license and disclaimer snippets for generated deliverables.

## Definition

- License snippets: attribution or redistribution text inserted into generated packets.
- Disclaimer snippets: limitation, usage, and "not legal advice" style notices for outward-facing artifacts.

## Warrant

Keep this folder when ai_ops workflows produce shareable artifacts that need consistent legal/disclaimer text.

## Trigger

Use this folder when a workbook/runbook includes external distribution, vendor content attribution, or
explicit disclaimer requirements.

## Not This Folder's Job

- Repository licensing for ai_ops source code.
- Secret storage.

Repo licensing remains at the root `LICENSE` and file-level headers.
